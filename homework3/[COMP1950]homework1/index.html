My Favorite Sites and Resources

W3 Schools

W3 Schools HTML site has descriptions and examples of many HTML and CSS related functions. The site is easy to navigate, comprehensive, and easy to understand.

Stylin' With CSS

Stylin' With CSS has a list of commonly used CSS properties, along with an explanatory desciption of each property and the values that can be used with it. This site nicley seperates the different properties into cataogries for an easier user experience, such as 'margin properties' and 'padding properties'.

DZone

DZone is a platform used by technology professionals including developers to gather up-to-date news and tools. The unique part is that the very professionals and developers that read DZone also have the oppurtunity to comment on articles and submit articles. Their catagories include news about Web Dev, Open SOurce, and Java

Stack Exchange

Stack Exchange is a forum and community site for developers and programmers. The subsets of communities it hosts include ones dedicated to topics such as web application development, software engineering, and user experience.

W3C HTML Validator

This site allows you to validate HTML markup.
